# Prose notes

- 인증 패널은 한 화면에서 한국어와 영어 장식 문구를 혼용하지 않는다. 고유 서비스명도 사용자 화면에서는 `챗지피티`, `오픈에이아이`처럼 한글 표기를 우선한다.
- 좌측 소개 영역은 로그인 상태를 반복 설명하지 않고 핵심 게임 흐름인 `광장 탐사와 자원 수집 → 탐사대원 교류 → 공동 건설과 항로 복구`를 짧게 설명한다.
- 기본 인증 행동은 `로그인`, `계정 생성`처럼 사용자가 예상하는 짧은 용어를 사용한다.
- 게임 소개는 온라인 익명성과 현실의 경계가 사람 사이의 대화를 어렵게 한다는 문제를 먼저 밝히고, 작은 행동과 대화를 공동 세계의 변화로 남겨 부담 없이 만나고 협력하게 한다는 해결을 설명한다.
